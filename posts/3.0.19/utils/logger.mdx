Remix PWA ships with a few amenities to make your development experience more enjoyable. One of these is a custom `logger` that
logs information and beig information to the browser console. This logger is only enabled in development environments so no need to worry
about it showing up in production.

## Usage

```js
import { logger } from '@remix-pwa/sw';

logger.log('This is a log message');
logger.info('This is an info message');
logger.warn('This is a warning message');
logger.error('This is an error message');
```
