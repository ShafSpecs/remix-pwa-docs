## Resources

### PWAs

- [Google's PWA Checklist](https://developers.google.com/web/progressive-web-apps/checklist): A great resource to make sure your PWA is up to snuff.
- [Web.dev's PWA Guide](https://web.dev/progressive-web-apps/): A great resource to learn more about PWAs in general. It contains a lot of stuffs like
cookbooks, guides, and more.

### Background Sync

- [A Good Summary](https://github.com/WICG/background-sync/blob/main/explainers/sync-explainer.md): A nice summary of what Background Sync is and how it works.

Contributions are welcome! If you have a resource you think should be added to this list, please open a PR to the `docs` branch of [this repository](https://github.com/ShafSpecs/remix-pwa-docs).
